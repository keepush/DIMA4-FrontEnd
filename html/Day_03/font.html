폰트

  • font-family: 글꼴 지정, 대중 글꼴을 사용할 것 권장
  • font-size: 글꼴 크기 지정
  • font-weightL 굵기 (100~900), 400(기본), bold, boler
  • font-varient: 영문자 기준, 소문자 크기로 대문자 표현

텍스트

  • color : 글자 색상
  • text-decoration : 밑줄 등을 넣거나 뺄 때
  • text-transform : 대소문자 변경
  • text-shadow : 그림자
  • white-space : 공백 처리
  • letter-spacing / word-spacing : 공백 처리
  • direction : 방향
  • text-align : 수평 정렬
  • text-justify : 정렬 시 남는 공백 처리
  • line-height : 줄 간격
  • text-overflow : overflow 속성과 함께 사용하고, 영역 내에 글자가 흘러넘칠 경우 표현 방법
  • (예) text-overflow
